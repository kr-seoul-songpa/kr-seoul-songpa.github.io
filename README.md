# 송파구 · 서울특별시

> 송파구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-songpa.github.io/**

송파구는 한강 남쪽 잠실벌과 백제 고도(古都)의 유산이 어우러진 서울 동남권의 자치구로, 555m 높이의 롯데월드타워와 석촌호수, 올림픽공원이 한데 모여 서울을 대표하는 랜드마크 밀집지로 꼽힙니다. 1988년 서울올림픽의 주 무대였던 이 땅에는 약 2천 년 전 한성백제의 왕성으로 추정되는 풍납토성과 몽촌토성이 그대로 남아, 초고층 마천루와 흙으로 쌓은 토성이 공존하는 독특한 도시 풍경을 만들어 냅니다. 잠실 관광·상업지구, 문정 법조단지, 국내 최대 규모의 가락농수산물종합도매시장이 각각 소비·업무·유통의 축을 이루며 경제적 활력을 뒷받침합니다. 인구 65만 명 안팎으로 서울 자치구 가운데 손꼽히는 규모를 자랑하며, 교육열이 높은 주거지와 대규모 아파트 단지가 넓게 자리합니다. 왕벚꽃이 흐드러지는 석촌호수와 몽촌토성 산책로는 계절마다 시민들의 발길을 끌어들이는 도심 속 휴식처입니다. 옛 송파나루와 송파장의 상업 전통이 오늘날 잠실 롯데월드몰과 가락시장으로 이어지며, 과거와 현재가 한 지역 안에서 두텁게 겹쳐 있는 곳입니다.
